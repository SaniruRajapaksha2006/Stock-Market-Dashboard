# 📈 Real-Time Stock Market Dashboard

Interactive dashboard for tracking real-time stock prices with technical indicators.

## 🚀 Features
- Real-time stock prices (NVDA, AAPL, MSFT, GOOGL, META, AMZN, TSLA)
- Multiple timeframes (1D, 1W, 1M, 6M, 1Y)
- Technical indicators (SMA, EMA, RSI, MACD, Bollinger Bands)
- Interactive candlestick charts
- Portfolio tracking with custom quantities
